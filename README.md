# game

Multiplayer game engine

Client:
* HTML5 (WebGL)
* Bootstrap
* JQuery
* Require.js
* Underscore.js
* Backbone.js
* Marionette.js

Server:
* Node.js
* Express.js (REST API)
* Mongoose.js (MongoDB)
